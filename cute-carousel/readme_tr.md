# Cute Carousel

Cute Carousel, React ve Tailwind CSS becerilerini geliştirmek için yapılmış **frontend pratik projesidir**. Basit bir carousel yapısı üzerinden temel UI ve state mantığı çalışılmıştır.

Bu projenin temel amacı **React state yönetimi ve UI etkileşimi pratiği yapmak** olup, karmaşık bir ürün veya tasarım sistemi oluşturmak değildir.

![Cute Carousel](./project-preview.png)
---

## Proje Amacı

Bu proje, React kullanarak **basit bir görsel carousel bileşeni geliştirmeye** odaklanır.

Amaçlanan pratik alanlar:
- Component state yönetimi
- Koşullu render mantığı
- Butonlarla kullanıcı etkileşimi
- Tailwind CSS ile temel responsive yapı

## Öğrendiklerim ve Yaptıklarım

- **React State Yönetimi**
  - `useState` ile aktif index kontrolü
  - State değişimine göre UI güncelleme

- **Component Mantığı**
  - Next / Previous butonları ile gezinme
  - Döngüsel carousel mantığı (sona gelince başa dönme)

- **Tailwind CSS Kullanımı**
  - Utility-first styling yaklaşımı
  - Responsive layout ve boşluk yönetimi
  - Hover ve active durumları

- **UI Düşünme Mantığı**
  - Basit ve kullanıcı dostu carousel yapısı kurma
  - Dinamik veri (görsel + açıklama) gösterimi
